# Ordner-Verschluesselungstool

- Ordner-Verschlüsselungstool für Windows, geschrieben in Java (mit Gradle)
- Das Tool durchläuft den gewünschten Ordner rekursiv und verschlüsselt alle enthaltene Dateien
